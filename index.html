2048 is a single-player puzzle where you merge equal-numbered tiles to reach 2048.

Sliding tiles in any direction doubles their value when merged.

The game is played on a 4x4 grid, so space is limited. Keep some free spots to maneuver and keep merging!

üì¢ How to play Numbers game 2048?

Use your arrow keys to slide the tiles in the direction you want. When tiles with equal numbers collide, they will merge into one tile and double in value.

üì¢Who created Numbers game 2048?

NumbersGame 2048 features the original 2048 game created by Gabriele Cirulli. He created the game in 2014 and was initially inspired by Veewo Studios' 1024, which was inspired by the game Threes.
